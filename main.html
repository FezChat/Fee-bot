Deployment Guide

1

Get the Source Code

Clone or download the bot from our GitHub repository:

git clone https://github.com/Fred1e/Fee-Xmd.git

Or download ZIP: Download ZIP

2

Choose Hosting Platform

Deploy on any of these platforms (click to deploy):

Heroku Koyeb Railway Render Replit
3

Configure Environment

Set up your environment variables:

PORT=3000
MONGODB_URI=your_mongodb_uri
SESSION_ID=your_session_id
PREFIX=!
MODS=your_number
CRON=0 */2 * * *
WELCOME=true
4

Install Dependencies

Install required packages:

npm install
5

Start the Bot

Run the bot with:

npm start

The bot will run on port 3000 by default. Scan the QR code that appears to connect your WhatsApp.
